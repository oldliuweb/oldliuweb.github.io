小资用户性价比最高的完美黑苹果配置

分享
2020-09-13

主要部件购买清单
1.处理器： 英特尔 Core i5-9600KF @ 3.70GHz 六核 1300元左右
2.主板： 华硕 TUF B360M-PLUS GAMING S 600元左右
3.显卡： 主显卡 AMD Radeon RX 580 ( 4 GB / AMD ) 某鱼400-500元左右
4.内存： 16 GB ( 玖合 DDR4 2666MHz ) 某多多389元
5.硬盘：建议购买NVME的 大小不等价格不一
6.网卡&蓝牙组件：某鱼或某宝搜索 BCM94360CD （双频1750M加蓝牙，用于免驱支持苹果的隔空传送）240元左右
EFI下载： https://www.lanzous.com/ia5b7wf

系统下载：
链接: https://pan.baidu.com/s/15iDpGzHcwJxt4zYPoZlbSw 提取码: wwcm

安装步骤：
1.设置主板BIOS，华硕主板bios上方有搜索，可以直接搜索我列出的选项修改保存即可。
2.制作系统U盘，用TransMac制作系统U盘。试用方法参见 https://blog.csdn.net/qq_36583249/article/details/86101076
3.替换EFI文件，试用PE或者DiskGenius，把U盘EFI分区的文件替换为上文提供的EFI文件，注意EFI分区下根目录有一个文件夹名称为EFI。
4.插入U盘，启动项选择该U盘的EFI启动项。
5.安装系统即可。

支持在线升级，今天在线升级到10.15.3无任何使用BUG，完美升级。

主板bios中更改（主板BIOS最上方有搜索按钮搜索后可直接修改）

Advanced —> Onboard Devices Configuration-Serial Port Configuration Serial Port —> Off
Advanced —> USB Configuration-XHCI Hand-off —> Enabled
Secure Boot -> OS Type -> Other OS
Advanced —> PCH configuration-IOAPIC 24-119 Entries —> disabled
CFG-LOCK -> disabled
Advanced -> Advanced PCH Configuration -> system time and alarm source -> Legacy RTC
