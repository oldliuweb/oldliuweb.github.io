Debian下安装CURL失败的解决方案

2020-09-13

Debian下需要安装curl程序可能会出现命令无效，结果但是通过

apt-get install curl

出现安装错误，看了很多资料，都是建议建议先做

apt-get update
apt-get upgrade

但是执行了，还是不行，最后在其他地方看到Debian 9.0以上需要安装

apt-get install sudo

执行了这个之后在执行安装curl就能正常安装了。

还有一个方法就是先执行

dpkg –configure -a

然后在执行
apt-get install curl

就可以正常安装了。

该方法实测有效！
